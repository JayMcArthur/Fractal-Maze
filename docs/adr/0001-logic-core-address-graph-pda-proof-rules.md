# Use Symbolic Address Logic With PDA Execution and Proof Rules

Fractal Maze Lab will use a logic core built around symbolic addresses, with address graphs and PDA-style stack execution as the first concrete strategy, plus optional proof rules for infinite descent and Cantor-style behavior. Raw PDA transitions are not the primary authoring format because they are awkward for editing and visual explanation, and pure stack execution is not sufficient for Cantor or Fractal Block Maze-style cases that may require symbolic parameters or coordinate-like addresses.
